# Breadcrumbs (`breadcrumbs`)

**Type:** slot · **Slot:** `breadcrumbs` · **Default:** enabled

Renders the breadcrumb trail in the header's context zone (see
[Header Navigation](../features/header-nav.md)) — the elastic zone that shows the
current tag hierarchy depth and grows with it. A thin wrapper around the shared
`Breadcrumbs` component; disabling the plugin drops breadcrumbs from the header, and
the context zone collapses to just the post count.
