# Tag Cloud (`tag-cloud`)

**Type:** slot · **Slot:** `home-explore` · **Default:** enabled

Renders the `ExploreBlock` component on the home page — a weighted cloud of tags used
to jump into the archive by topic or location. Mounted into the `home-explore` slot;
disabling the plugin drops the block from the home page entirely, leaving just the post
grid and whatever other slot plugins ([`timeline`](timeline.md),
[`distraction-free`](distraction-free.md)) remain enabled.
