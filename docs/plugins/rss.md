# RSS (`rss`)

**Type:** service · **Routes:** `/feed.xml`, `/feed` · **Default:** enabled · **Title:** RSS

RSS 2.0 feed auto-generated from published posts, reflecting the blog title/description
settings, cached for one hour. Disabling the plugin 404s both routes and removes feed
`<link>` references from served pages.

See [Syndication & SEO](../features/syndication.md) for the sitemap and robots.txt
features that ship alongside it (those are not plugins — always on).
