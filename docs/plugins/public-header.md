# Public Header (`public-header`)

**Type:** slot · **Slot:** `header` · **Default:** enabled

Renders the public site's header shell — the four-zone row (identity, context, nav,
tools) described in [Header Navigation](../features/header-nav.md) — into which the
other header-related slot plugins ([`nav-menu`](nav-menu.md),
[`breadcrumbs`](breadcrumbs.md)) mount their content. Disabling this plugin removes the
header entirely from public pages.
