# Slideshow (`slideshow`)

**Type:** slot · **Slot:** `slideshow` · **Default:** enabled

An auto-advancing slideshow for the media viewer, injected via the `slideshow`
sub-slot when a post has more than one media item. Adds a top-right play/pause toggle
next to close/share plus a bottom control bar. Disabling the plugin removes the
slideshow controls; the viewer falls back to manual navigation only.

See [Immersive Mode & Media Viewer](../features/immersive.md#slideshow) for behavior
details.
