# Admin Home (`admin-home`)

**Type:** route · **Routes:** `/light` · **Area:** `admin-home` (core) · **Default:** enabled

The admin dashboard (`DashboardPage`) — the compose-first daily landing page: primary
nav, editor essentials, quick actions. Own single-plugin `Core` area, so it can never
be disabled; without it the admin app would have no entry point.

See [Admin UX](../features/admin-ux.md) for the design principles behind the dashboard.
