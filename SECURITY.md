# Security Policy

## Supported Versions

Please note that this is a rapidly evolving project. Only the latest commit on the main branch is considered supported for security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them by sending an email to the repository owner: `enrich-rare-posing@duck.com`.

You can also use GitHub Security Advisories to privately report vulnerabilities if enabled for this repository.

We will try to acknowledge receipt of your vulnerability report as soon as possible and address it accordingly.
